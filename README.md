# Shri Harivansh Reader

A lightweight Next.js reader shell for devotional content.

## What is included

- App Router structure
- Home screen for selecting a reading collection
- Verse-by-verse reader view
- Audio player per section
- Font size controls
- Local progress persistence through `localStorage`
- Clean structure that can later plug into Firebase authentication and sync

## Run locally

```bash
npm install
npm run dev
```
